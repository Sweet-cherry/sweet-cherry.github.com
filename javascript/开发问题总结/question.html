        # ios动画卡顿问题

        在一次做卡片翻转动画过程中，发现在ios上背面div显示不出来， 解决方式：添加css样式：

        -webkit-backface-visibility: hidden;
backface-visibility: hidden

        使用transform: rotateY(180deg)等3d动画 不要写在最外层div盒子上，会使动画卡顿，需要里层的盒子有单独的过渡效果，比如

        <div class="card">
    <div class="card-side front">正面</div>
    <div class="card-side back">反面</div>
</div>

        transition动画样式：

        .card{
    transform: translate3d(0,0,0);
    position: relative;
    width: 100px;
    height: 100px;
}
.card-side{
    position: absolute;
    width: 100%;
    height: 100%;
    transition: all .5s;
    backface-visibility: hidden;
}
.card-side.front{
    transform: rotateY(0)
}
.card-side.back{
    transform: rotateY(180deg)
}
/* 翻转动画 */
.card.rotate .front{
    transform: rotateY(180deg)
}
.card.rotate .back{
    transform: rotateY(0deg)
}
